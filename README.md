# Othello Agent

## 概要
本リポジトリは、Othello（リバーシ）を対象とした自己学習エージェントの構築・評価環境を提供します。モデルは自己対戦データを用いて逐次的に性能向上を図り、対戦結果の統計的指標によって客観的に評価します。

## データ生成プロセス
- **自己対戦ベース**：エージェント同士の対局をシミュレートし、ゲームログをデータセットとして蓄積します。  
- **勝者ステップ抽出**：各対局において勝者側の手番のみをサンプリング対象とし、勝者の戦略情報を学習素材とします。  
- **部分抽出方針**：勝者側全手番を使用せず、代表的な一部手番のみを抽出。  
  - 初期段階では相手が極端に弱体であるため、全手番を学習させても後続の高性能相手に対抗できる保証はない。  
  - データ量を削減することでトレーニング時間を短縮し、学習効率を向上させます。

## 対局相手構成
- **完全モデル制御**：一方のプレイヤーは学習済みモデルの出力に完全従属し、決定論的に手番を選択します。  
- **ハイブリッド制御**：もう一方は一定比率でランダムポリシーとモデル予測を混合し、確率的に手番を決定します。  
  - モデルが成熟するにつれ、相手が完全ランダムの場合、勝率が低下しデータ取得が困難になるため、探索的なランダム要素を保持します。

## インストール手順
```bash
npm i
```
Node.js (v20.x) と npm (v6.x) が前提環境です。

## トレーニング実行
```bash
npm run train:othello
```
自己対戦を繰り返しながら、抽出された勝者手番をバッチ学習に供給します。

## ゲーム体験コマンド
```bash
npm run game:othello
```
上記コマンド実行後、ブラウザで `http://127.0.0.1:9090` にアクセスすると、Web UI 上で対局を操作できます。

## テスト・評価
```bash
npm run test:othello
```
モデル対ランダムエージェントの勝率を数値で出力し、性能を直感的に把握できます。

## 人間プレイによるトレーニング加速
ローカル対局で人間が実行した有効手は自動的にトレーニングデータとして取り込まれ、自己対戦のみの場合に比して学習効率が顕著に向上します。人間が勝利した手番は即座にサンプルとして蓄積されます。

## 推奨ハードウェア構成
- CPU: Apple M2 (MacBook)  
- メモリ: 24 GB 以上  
- OS: macOS (他のプラットフォームでも Node.js が動作すれば可)
