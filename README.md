# Othello AI トレーニングフレームワーク  

## 概要  
本リポジトリは、Reversi（Othello）に特化した自己教師あり学習パイプラインを実装します。エージェントは自己対戦により自律的にデータを生成し、勝利側の選択手を抽出してモデル更新に利用します。

## データ生成プロトコル  
- **自己対戦生成**：エージェント同士が対局し、結果に基づくサンプルを収集。  
- **勝者手抽出**：勝利側の着手列から、全体ではなく選択的に抜粋した局面のみをトレーニングサンプルとします。  
  - **抽出根拠**  
    1. 初期段階の相手は学習能力が未熟であるため、勝者全手を模倣しても上位相手に対抗できないケースが多発。部分的な模倣に留めることで、汎用的な戦術形成が促進されます。  
    2. 抽出量を限定することで、バッチ処理時間が削減され、トレーニングスループットが向上します。

## 対戦相手構成  
- **純粋モデル制御**：一方のエージェントは完全に学習済みモデルに依存し、ポリシー決定を行います。  
- **ハイブリッド制御**：もう一方は、一定確率でランダム手とモデル予測手を混合し、探索的行動を導入します。  
  - **設計意図**：エージェントが成熟すると、全ての手がランダムであってもモデルが容易に優位を確保できなくなり、結果として有意義な学習サンプルが枯渇します。ハイブリッド戦略により、データ多様性と難易度を維持します。

## インストール手順  
```bash
npm i
```
- **推奨環境**：Node.js v20 系、npm v6 系

## トレーニング実行  
```bash
npm run train:othello
```
本コマンドは上記プロトコルに従い、自己対戦データを自動取得・前処理し、TensorFlow.js ベースのネットワークを最適化します。

## ゲーム体験 & データ収集  
```bash
npm run game:othello
```
起動後、ブラウザで `http://127.0.0.1:9090` にアクセスすると対局 UI が表示されます。ユーザーが勝利した手番は自動的に学習データセットへインクリメンタルに追加され、次回トレーニング時に反映されます。人間プレイヤーの高品質手を直接投入することで、トレーニング効率が顕著に向上します。
